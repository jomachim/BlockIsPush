## How to play ?
use keyboard :
- (azerty) :
  - <code>Z</code> for UP
  - <code>S</code> for DOWN
  - <code>Q</code> for LEFT
  - <code>D</code> for RIGHT
  - <code>A</code> for REWIND
- (qwerty) :
  - <code>W</code> for UP
  - <code>S</code> for DOWN
  - <code>A</code> for LEFT
  - <code>D</code> for RIGHT
  - <code>Q</code> for REWIND
